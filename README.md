# Car Rental

A simple simulated car rental system implemented using object-oriented design principles.

## Requirements

The system supports:

- Reserving a car of a selected type for a specified start date and time.
- Reserving a car for a given number of days.
- Three car types:
    - Sedan
    - SUV
    - Van
- A limited number of cars available for each type.
- Prevention of reservations when no car of the requested type is available.
- Unit tests demonstrating that the requirements are satisfied.

## Technology

- Java 25
- Maven
- JUnit 5

## Running the Tests

```bash
mvn clean test
```
